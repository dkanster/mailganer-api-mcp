# Mailganer REST API — обзор

Источник: [https://mailganer.com/documentation/api/](https://mailganer.com/documentation/api/)

## Документация по REST API для Mailganer

Документация по REST API для Mailganer

Методы поделены на несколько групп:
1. Работа с личным кабинетом Mailganer. Стандратные методы по работе с подписчикам, списками, рассылками.
2. Интеграция Mailganer со сторонними приложениями. Группа методов называется "Работа с пользователями"

Напишите нам на api@mailganer.com , если вам не хватает какого то метода.

## Авторизация

1. В методах, содержащих в url v1, авторизация происходит через использование апи ключа в теле запроса в переменной api_key .
2. В методах, содержащих в url v2, авторизация происходит через использование апи ключа в заголовке Authorization: CodeRequest {{api_key}}

API ключ находится в разделе «Настройки аккаунта».

Обращаем внимание, что методы по работе с SMTP задокументированы отдельно и имеют свою отдельную авторизацию.

## Лимиты

К API можно обращаться не чаще 500 раз в минуту. В случае превышения лимита, вы получите ошибку с кодом 429. Напишите нам, если требуется увеличить лимит.

## Пагинация

Некоторые методы отдают по 25 записей на странице. В переменной count находится количество записей. Ответ находится в json-масиве result. В переменных next и previous находятся ссылки на следующую или предыдущую страницы. Если следующей или предыдущей страницы нет, то вернётся значение null.

```json
{
 "count": 2,
 "next": null,
 "previous": null,
 "results": [
 {
 "source": 1,
 "title": "user_var_float",
 "type": 2,
 "type_text": "Вещественное число"
 },
 {
 "source": 1,
 "title": "user_var_datetime",
 "type": 4,
 "type_text": "Дата и время"
 }
 ]
}
```
